# Manual Installation <br/>

1. **Download theme.zip in releases** <br/>

2. **Execute**  _"spicetify path userdata"_ <br/>

Example of output:<br/>
```
>_ C:\Users\pepegog\AppData\Roaming\spicetify
```
3. __Navigate to__ *\<output\>\themes*
  
5. __Unarchive theme.zip here__ <br/>

6. __Type these commands in cmd__ <br/>
```
spicetify config current_theme wine-night color_scheme dark
spicetify config inject_css 1 replace_colors 1 overwrite_assets 1
spicetify apply
```  
__*Done! Enjoy*__  
> [!WARNING]
> Theme has Light scheme.
>
> However, i dont recommend you to use it, as some buttons blend into background.
